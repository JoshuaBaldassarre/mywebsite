About

The aim of mywebsite is to give anyone a good looking basic website template for free, general, usage.
Mywebsite is open source and free for anyone to use as long as the mywebsite. brand or original creator/s are credited correctly and clearly.

I created this website template because I could never find a template that was simple for beginners and didn't involve tones of third party libraries that make the website more complicated than it needs to be. So I decided to set out and learn basic CSS and HTML to create a good looking and fully functional website template, with the simplest possible code for beginners trying to make their own website.*

This website was made completely voluntarily in my free time, donating here to support and motivate further development for this project would be greatly appreciated.
